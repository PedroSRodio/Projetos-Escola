Dicionário de fundamentos da informática:


API (Application Programming Interface): Conjuntos de instruções e padrões para interação entre diferentes softwares.


Armazenamento (SSD - Solid State Drive): O armazenamento SSD é uma tecnologia de armazenamento de dados que utiliza memória flash para acesso rápido e eficiente aos dados. Os SSDs são mais rápidos e confiáveis do que os discos rígidos tradicionais (HDDs) devido à ausência de partes móveis. Eles são usados para armazenar o sistema operacional, programas e arquivos de uso frequente para melhorar o desempenho do computador.


Arquitetura de Computadores: A estrutura interna de um computador, incluindo CPU, Memória e periféricos.


Assembly Language: Linguagem de baixo nível usada para programar diretamente o hardware.


Cloud Computing: Oferecimento de serviços de computação pela internet, permitindo acesso remoto a recursos computacionais.


Compiladores e Interpreters: Ferramentas que traduzem código fonte em instruções executáveis.


Concurrency: A capacidade de um sistema lidar com múltiplas tarefas simultaneamente.


Criptografia: Técnica de codificação de dados para proteger sua privacidade e segurança.


Data Structures: Formas organizadas de armazenar e gerenciar dados em computadores.


Database Management Systems (DBMS): Softwares para gerenciar e manipular bancos de dados.


Estrutura de Dados: Formas de organizar e armazenar dados em um computador para facilitar o acesso e a manipulação.


Fonte de alimentação:A fonte de alimentação fornece energia elétrica para todos os componentes do computador. Ela converte a corrente alternada da tomada em corrente contínua utilizada pelos componentes internos. Uma fonte de alimentação de qualidade e com capacidade adequada é essencial para garantir o funcionamento estável e confiável do computador.


Gabinete: O gabinete é o invólucro que abriga todos os componentes internos de um computador. Ele fornece proteção aos componentes, facilita o gerenciamento dos cabos e possui portas para conectar dispositivos externos, como monitores, teclados e dispositivos de armazenamento.


Hardware: Os componentes físicos de um sistema computacional, como processador, memória RAM e dispositivos de armazenamento.


Machine Learning: Ramo da inteligência artificial que permite aos sistemas aprenderem e melhorarem com base em dados.


Memória RAM (Random Access Memory): A memória RAM é um tipo de memória volátil usada pelo computador para armazenar temporariamente dados e instruções que estão sendo usados ativamente. Quanto mais RAM um computador possui, mais programas e processos podem ser executados simultaneamente sem prejudicar o desempenho.


Monitor: O monitor é o dispositivo de saída que exibe informações visuais geradas pelo computador. Existem diferentes tipos de monitores, como LCD, LED, OLED, entre outros, com diversas resoluções e tamanhos. A qualidade do monitor influencia diretamente a experiência visual do usuário.


Placa de vídeo: A placa de vídeo, ou GPU (Unidade de Processamento Gráfico), é responsável pelo processamento de gráficos e imagens em um computador. É essencial para jogos, edição de vídeo, design gráfico e outras tarefas que exigem grande capacidade gráfica. Algumas placas de vídeo também podem acelerar tarefas de computação intensiva, como renderização 3D e aprendizado de máquina.


Placa-mãe: A placa-mãe é a peça central de um computador, conectando todos os componentes do sistema.


Processador(CPU):O processador, também conhecido como CPU (Unidade Central de Processamento), é o cérebro do computador. Ele executa instruções e realiza cálculos necessários para executar programas e operações do sistema. A velocidade e o número de núcleos de um processador afetam diretamente o desempenho geral do computador.


Protocolos: Conjunto de regras para comunicação em redes de computadores, como TCP/IP.


Redes de Computadores: Sistemas que permitem a comunicação e o compartilhamento de recursos entre dispositivos conectados.


Redundancy: A prática de ter sistemas ou dados de backup para garantir a disponibilidade e segurança.


Security Protocols: Métodos e técnicas para proteger sistemas e dados contra ameaças.


Sistemas Operacionais: Software que gerencia recursos de hardware e fornece uma interface para usuários e aplicativos.


Software: Os programas e aplicativos que operam em um sistema de hardware para realizar tarefas específicas.


Web Development: O processo de criar aplicativos e sites para a internet.
