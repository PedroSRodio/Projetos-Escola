Dicionário de Desing Digital:


UI(User Interface): UI refere-se à interface visual com a qual os usuários interagem.


UX(User Experience): Ux abrange a experiência geral do usuário ao interagir com o produto.


Wireframe: Um esboço visual que mostra a estrutura básica de um site ou aplicativo, sem focar em detalhes visuais.


Protótipo: Uma versão inicial e funcional de um projeto digital, usada para testar ideias e funcionalidades antes da implementação final.


Responsive Design: Design que se adapta a diferentes dispositivos e tamanhos de tela para garantir uma boa experiência do usuário em qualquer contexto.


Mockup: Uma representação estática de um design, geralmente mais detalhada do que um wireframe.


Grid System: Um sistema de grade que ajuda a organizar elementos de design de forma consistente


Typography: O uso de fontes e texto de maneira eficaz no design.


Color Theory: O estudo das cores e sua aplicação no design para transmitir mensagens específicas.


User Persona: Uma representação fictícia do usuário típico de um produto, usado para orientar o design.


A/B Testing: Teste comparativo de duas versões diferentes de um design para determinar qual é mais eficaz.


Wireframing Tools: Ferramentas de software usadas para criar wireframes.


Grid Layout: Uma estrutura de grade usada para organizar elementos de design de forma consistente.


UI Elements: Componentes visuais, como botões, menus e ícones, em uma interface de usuário.


Visual Design: A parte estética do design, incluindo cores, imagens e estilo visual.


Interaction Design: Testes realizados com usuários reais para avaliar a usabilidade de um design.


Information Architecture: A organização e estruturação de informações em um sistema digital.


Accessibility: A prática de tornar produtos digitais acessíveis a pessoas com deficiências.


Motion Design: O uso de animações e transições para melhorar a experiência do usuário.


Design Systems: Conjuntos de padrões e diretrizes para garantir consistência no design.
