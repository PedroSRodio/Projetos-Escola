Dicionário de Técnicas de Programação e Algoritimo:


Algoritimo: Um conjunto de instruções passo a passo para resolver um problema ou realizar uma tarefa.

Arrays: Estruturas de dados que armazenam coleções ordenadas de elementos do mesmo tipo.


Árvores: Estruturas de dados hierárquicas com um nó raiz e nós filhos, usadas em algoritmos de busca e organização de dados.


Bit Manipulation: A manipulação de bits individuais em dados binários.Complexidade de Algoritmos e Notação O (Big O): Medidas de desempenho e eficiência de algoritmos, indicando o tempo e o espaço necessários para sua execução.


Dynamic Programming: Uma abordagem para resolver problemas dividindo-os em subproblemas menores.


Estruturas de Controle: Instruções que controlam o fluxo de execução de um programa, como condicionais (if-else) e loops.


Funções: Blocos de código que realizam uma tarefa específica e podem ser chamados várias vezes em um programa.


Graph Algorithms: Algoritmos usados para resolver problemas em grafos, como o algoritmo de Dijkstra.


Hashing: O processo de mapear dados de tamanho variável para dados de tamanho fixo.


Listas: Estruturas de dados que podem conter elementos de diferentes tipos e podem crescer ou diminuir dinamicamente.


Loops: Instruções que repetem um bloco de código até que uma condição seja atendida.


Matrizes: Arrays multidimensionais, organizados em linhas e colunas.


Modularização: A prática de dividir um programa em módulos ou funções independentes.


Object-Oriented Programming (OOP): Um paradigma de programação baseado em objetos e suas interações.


Operadores: Símbolos usados em expressões para realizar operações matemáticas, lógicas ou de comparação.


Ordenação e Busca: Algoritmos para organizar elementos em uma sequência específica ou encontrar um elemento em uma coleção de dados.


Pilhas e Filas: Estruturas de dados que seguem o princípio de "último a entrar, primeiro a sair" (pilha) ou "primeiro a entrar, primeiro a sair" (fila).


Pointer: Um tipo de variável que armazena o endereço de memória de outra variável.


Recursion: A técnica em que uma função chama a si mesma para resolver problemas.


Recursividade: Técnica em que uma função chama a si mesma para resolver problemas de forma iterativa.


Software Development Life Cycle (SDLC): O processo de desenvolvimento de software, que inclui etapas como análise, design, implementação e teste.


Tipos de Dados: Formas específicas de dados que podem ser manipulados por um programa, como inteiros, strings e booleanos.


Variáveis: Locais de armazenamento de dados em um programa, como números, texto ou valores booleanos.
